Ansible Modules

  1. copy :Copy files from source location to destination location
       src: index.html
       dest: /var/www/html/index.html
  2. service :TO restart a service
       name: name of a service(httpd)
       state: started
        enabled: yes
  3. user: To create a user
      name: naga
      state: present
  4. user: To create a group
      name: wheel (group name)
      state: present
  5. user: To add user to a group
      name: naga
      group: present
    getent group ->to know list of groups on ubuntu
    getent group sudo -> to know list of users belong to sudo group
  6. debug:looping through list of items
       msg: "{{ item }}"
       with_items: "{{ fruits }}"
  7. debug:
       var: version
  8. apt: To install any software on ubuntu
      name: s/w name(git)
      state: present
       when: ansible_os_family == 'Debian'
  9. yum: To install any software on celtos
      name: s/w name(git)
      state: present
       when: ansible_os_family == 'RedHat'
