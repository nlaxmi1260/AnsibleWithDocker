Ansible Variables

  1. Variables declaration
    vars:
       package_name: docker.io
       is_useful: true
       age: 35
       fruits:
        - apple
        - orange
        - mango
        - pineapple
       members:
         mallikarjun: husband
         dinesh: son
         gagana: daughter
  2. Accessing variables
    {{ package_name }}
    {{ fruits }}
    {{ fruits[1] }}
    {{ members }}
    {{ members.mallikarjun }}
    {{ age }}
  3. Variable file reference in main playbook
    vars_files:
       - filename.yml
  4. In Variable file there should not be vars: section
    name: naga
    age: 35
  5. Providing variable value at runtime
    ansible-playbook /workingdir/Practice/Variables/jinja2.yml --extra-vars '{"version":"1.0"}'
  6. Providing variable file at runtime
    ansible-playbook /workingdir/Practice/Variables/jinja2.yml --extra-vars '@myvars.yml'
  7. precedence variable given at runtime will get the heighest precedence
    $ ansible-playbook /workingdir/Practice/Variables/jinja2.yml --extra-vars '{"version":"1.0","another_var":"runtime value"}'
